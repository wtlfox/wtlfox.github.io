获取当前定位

通过调用 getCurrentPosition() 函数获取用户当前定位位置。异步地请求获取用户位置，并查询定位硬件来获取最新信息。

getCurrentPosition方法有三个参数

window.navigator.geolocation.getCurrentPosition(successCallback, errorCallback, options);

  • 第一个参数成功时调用的回调函数

  • 第二个参数失败时调用的回调函数

  • 第三个参数设定最长可接受的定位返回时间、等待请求的时间和是否获取高精度定位

css盒子等比例缩放

padding-bottom有一个非常重要的特性，即当它的值是百分比形式时，
百分比的基数是其所在元素的父元素的宽度，而不是高度（同padding-left和padding-right一样），
则具体的解决方案是：将元素的高度height设为0，
使其高度等于该元素的padding-bottom，再合理设置该元素padding-bottom的值

例子：

将其每个div的宽度设为25%，高度设为0，让其左浮动，
padding-bottom的值设为25%（即父元素body的宽度的25%），
这样便成为了四个水平方向排列的正方形，并且在你改变浏览器窗口大小时，
同样会等比例缩放

js兼容性问题

JS的兼容性问题（总结）:

1.注册事件的兼容处理
 if(obj.addEventListener){
    obj.addEventListener(type,handler,false);
  }else{
    obj.attachEvent('on'+type,handler);
  }
  2.解除绑定的事件兼容处理
  if(obj.removeEventListener){
    obj.removeEventListener(type,handler);
  }else{
    obj.detachEvent('on'+type,handler);
  }
3.阻止冒泡的兼容处理
  if(event.stopPropagation)
   {
    //标准浏览器
     event.stopPropagation();
   }else{
    //ie6,7,8
     event.cancleBubble = true;
   }
4.阻止默认行为的兼容处理
  if(e.preventDefault){
    //标准浏览器
    e.preventDefault();
   }else{
    //IE678
    e.returnValue = false;
   }
5.事件对象的兼容处理
  e = e || window.event;
6.获取触发事件的对象
  var tar = e.target || e.srcElement;

angular初识

Angular的核心特性

  • MVC
  • 模块化
  • 自动化双向数据绑定
  • 指令系统

下载angular(两种方式任选其一)

  • bower install angular –save
  • npm install angular –save

重要的指令

  • ng-app

    • 所有需要ng管理的代码必须被包裹在一个有ng-app指令的元素中
    • ng-app是ng的入口，表示当前元素的所有指令都会被angular管理（对每一个指令进行分析和操作）
    • 当angular执行ng-app的代码时会开辟一块空间，这块空间用来存储页面上可能会用到的数据模型

  • ng-model是双向数据绑定的指令

Angular特点

  • 最大程度的减少页面的DOM操作

示例

    <input type="text" ng-model="user.name" placeholder="请输入用户名">
    <p>hello <strong>{{user.name}}</strong></p>
      </div>
 <script src="../node_modules/angular/angular.min.js"></script>
</body>

面向对象之原型

原型链

  • 什么是原型链？
    • 原型链是由对象组成的有限对象链，用于实现继承和共享属性。
  • 原型链的作用？
    • 实现数据共存、继承
  • 原型链的本质？
    • 实例对象与原型形成的链，链之间的联系通过proto。
  • 没有办法通过实例修改原型上的内容。
  • 当要找的某个属性找到了，就不会再向上找了，如果没找到就会按着原型链继续往上找直到找到。

继承的思想：别人给了我，我的还是我的

初识面向对象

什么是面向对象？

  • 面向对象是一种编程模式

    • 好处：1.可维护性高 2.重用性高

  • 构造函数、实例对象、原型三者的关系？

    • 所有的构造函数都有一个原型属性prototype,属性值也是一个对象（new Object()）原型对象
    • 实例对象都有一个默认的属性__proto__，该属性不是标准（每个浏览器的这个属性的名
      称不一样）在实际开发中不可以使用这个属性，该属性是浏览器自己使用，该属性的指向和原
      型prototype指向相同。
      • 3.原型对象（prototype）里面都有一个constructor属性，该属性指向原型所属的构造函数。

  • 原型

    • 实现数据共享
    • 构造函数中原型属性prototype指向的区域被所有的实例共享
    • prototype是从构造函数的角度来说的。
    • __proto__是从实例对象的角度来说的
  • this
    • 构造函数中的this
    • 原型方法中的this
    • 事件函数中的this

  • new 实际上背后做了什么？

    • 分配内存，用来存储具体的数据。
    • 分配好之后用this指向这块区域（this存储内存地址）实际上this就是一个内存地址
    • 接下来会通过this指向的这块区域添加属性，并且存放数据。
    • 返回this(内存地址)

总结

一般情况下，方法放在原型上，属性放到构造函数中，达成的效果是：所有实例共享方法，但数据各自有一份。

动画

2D转换transform

  • 位移 translate()
    • transform:translateX(600px)向右平移600px
    • transform:translateY(-400px)向上移动400px
    • transform:translate(200px -200px)向右移动200px 向上移动200px
  • 旋转 rotate()
    • transform:rotate(45deg)顺时针旋转45度
    • transform:rotateX(45deg)顺时针绕着x轴旋转
    • transform:rotateY(45deg)顺时针绕着y轴旋转
    • 规律：当值越来越大的时候，绕着顺时针旋转，当值越来越小的时候，逆时针旋转
  • 缩放 scale()
    • transform:scale(2)放大2倍
    • transform:scale(0.1)缩小到0.1被
    • 规律：值大于1时放大，下于1时缩小
  • 倾斜 skew()
    • transform:skewX(45deg)角度为正，向左倾斜，角度为负，向右倾斜
    • transform:skewY(-45deg)角度为正，向下倾斜，角度为负，向上倾斜
  • transform-origin转换原点（指转换元素以哪个点进行转化）

    3D转换

  • 3d位移 translate
    • translateX 向右为正，向做为负
    • translateY 正值向下，负值向上
    • translateZ 正值靠近我们，负值远离我们

      透视perspective

  • 脑显示屏是一个2D平面，图像之所以具有立体感（3D效果），其实只是一种视觉呈现 ，通过透视可以实现此目的。
  • 透视可以将一个2D平面，在转换的过程当中，呈现3D效果。（没有perspective，便“没有”Z轴）
  • 透视原点 perspective-origin
    • Perspective-originj:x y
    • X轴代表眼睛在水平动
    • Y轴代表眼睛在竖直动
  • Perspective可以写在父级，也可以写在自己
    • 父级的时候 是一个属性 perspective perspective:1000px;
    • 自己的时候是一个属性值 perspective()给transform
  • 转换方式 transform-style
    • transform-style:preserve-3d;必须写在父级元素

      过渡 transition

  • 过渡属性 transition-property
  • 过渡持续时间 transition-duration
  • 过渡延迟 transition-delay
  • 过渡时间函数 transition-timing-function

    动画 animation

    -定义动画
    • @keyframes 动画名｛from{} to{}｝
    • 使用动画
      • 动画名称 animation-name
      • 动画持续时间 animation-duration
      • 动画延迟 animation-delay
      • 动画时间函数 animation-timing-function
      • 动画循环次数 animation-interation-count （infinite无限次循环）（steps(步数)分几步完成）
      • 动画方向 animation-direction
        • normal 正常播放
        • alternate 动画应该轮流反向播放
      • 动画播放状态 animation-play-state
        • paused 动画停止
        • running 动画播放
      • 动画执行后的状态 animation-fill-model
        • forwards 保持最后完成的状态
        • backwards 回到初识状态
    • Animation:动画名称 动画时间 动画延时 其他任选

伸缩布局及盒子尺寸

盒子尺寸box-sizing

  • box-sizing:content-box(默认值)
    • 盒子总宽度=width+padding+border
  • box-sizing:border-box
    • 让盒子的总宽度等于设置的width

      Flex布局（伸缩布局）

  • 伸缩容器display:flex
    • 伸缩容器属性
      • flex-direction 调整主轴的方向（默认是水平方向）四个值可选：
        • row (水平方向左到右)
        • column (竖直方向上到下)
        • row-reverse (水平反向)
        • column-reverse (竖直反向)
      • flex-wrap 调整伸缩项目是否换行 两个值可选 ：nowrap(不换行默认)，wrap(换行)
      • align-items 调整伸缩项目在侧轴的对齐方式 四个值可选：
        • flex-start 侧轴起始点对齐
        • flex-end 侧轴终点对齐
        • center 侧轴居中对齐
        • stretch 伸缩对齐（默认）
      • align-content 调整换行后伸缩项目的对齐方式 六个值可选：
        • flex-start
        • flex-end
        • center
        • space-around 四周环绕
        • space-between 两端对齐
        • stretch 拉伸
      • justify-content 调整伸缩项目在主轴方向上的对齐方式，五个值可选：
        • flex-start 主轴起始点对齐
        • flex-end 主轴终点对齐
        • center 居中对齐
        • space-around 四周环绕
        • space-between 两端对齐
      • flex-flow 是flex-direction和flex-wrap的简写形式
  • 伸缩项目
    • 伸缩项目属性
      • flex 代表伸缩项目在伸缩容器中所占的剩余位置的比例
      • align-self 设置自己的对齐方式 会覆盖父元素设置的侧轴对齐方式
      • order 设置伸缩项目的排序方式 数字越大越靠后，越小越靠前

阴影属性及颜色表示法

阴影

  • 文本阴影
    • text-shadow:水平偏移 竖直偏移 模糊程度 阴影的颜色
    • 水平偏移：值越大，越往右，值越小，越往左
    • 竖直偏移：值越大，越往下，值越小，越往上
    • 模糊：模糊不能为负值，值越大，越模糊
    • 阴影颜色：为你想要的颜色
    • text-shadow:10px 10px 0px #00ff000
  • 盒子阴影
    • box-shadow:水平偏移 竖直偏移 模糊程度 模糊的扩展距离 阴影颜色 内嵌
    • 模糊的扩展距离：从元素边开始到开始模糊的地方，值越大扩展的距离越大
    • 内嵌：让阴影变成内嵌的阴影，称之内阴影，不写默认是外阴影
    • box-shadow:10px 10px 1px 10px #ccc inset

      边框

  • 边框圆角
    • border-radius
    • 值有四种情况
      • 一个值，代表四个角的圆角值都一样border-radius:10px
      • 两个值，代表对角线的两个角的值（左上/右下 右上/左下）border-radius:10px 20px
      • 三个值，代表左上，右下，右上/左下 border-radius:10px 20px 30px
      • 四个值，代表顺时针各个角border-radius:10px 20px 30px 40px

颜色表示方式

  • rgba()
    • r:red
    • g:green
    • b:blue
    • a:alpha 不透明度（0-1）
    • background-color:rgba(125,125,125,0.5)
  • hsla()
    • h:色调
    • s:饱和度
    • l:亮度或明度
    • a:alpha不透明度(0-1)
